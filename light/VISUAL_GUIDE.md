## 🛒 Making a Sale (POS Screen)

### Step-by-Step:

**1. Click "POS" at the top**

**2. You'll see two sides:**

```
┌─────────────── POS ───────────────┐
│                                   │
│  Products              Cart        │
│  ┌──────────┐        ┌─────────┐  │
│  │ Mouse    │        │ Empty   │  │
│  │ $25.99   │        │         │  │
│  │ Click me │        │         │  │
│  └──────────┘        └─────────┘  │
│  ┌──────────┐                     │
│  │ Keyboard │        Total: $0.00 │
│  │ $35.99   │                     │
│  └──────────┘        [ Complete ] │
└───────────────────────────────────┘
```

**3. Click on products** to add them to cart
**4. Enter payment amount**
**5. Click "Complete Sale"**
**6. Done!**

## 📦 Adding a Product (Inventory Screen)

### Step-by-Step:

**1. Click "Inventory" at the top**

**2. Click "+ Add Item" button**

**3. Fill in the form:**

```
┌─────── Add New Product ───────┐
│                               │
│  Product Name: [ Mouse      ] │
│  SKU:          [ MOUSE-001  ] │
│  Price:        [ 25.99      ] │
│  Cost:         [ 15.00      ] │
│  Quantity:     [ 50         ] │
│                               │
│         [ Create Item ]       │
└───────────────────────────────┘
```

**4. Click "Create Item"**
**5. Done!**

## 🆘 Help! What Do I Click?

### I Want To...

**Start the program**
→ Double-click `LightERP_Launcher.exe`

**Make a sale**

→ Stop the program first

## 🎨 Understanding the Colors

**Green buttons** = Start, Create, Complete, Success  
**Red buttons** = Stop, Delete, Cancel  
**Blue buttons** = Information, View, Update  
**Yellow/Orange** = Warning, Low Stock, Review  

---

## 🔢 The Numbers

### On Dashboard:

**Total Sales** = Money you made from selling  
**Balance** = Current money in business  
**Inventory Value** = Worth of products you have  
**Low Stock** = Products running out (need to reorder)

## ⚠️ Don't Panic If...

### "Browser doesn't open"
- Open browser yourself
- Type: `localhost:8005`
- Press Enter

### "Can't find the file"
- Look for `LightERP` folder
- Look for file ending in `.exe`
- Double-click the one with "Launcher" in name

### "Numbers look wrong"
- Check date range in Reports
- Make sure you're looking at right tab
- Try clicking "Refresh" in browser

### "Made a mistake"
- Don't worry! You can delete transactions
- Your data is backed up (if you followed backup steps)
- Close and restart if needed
